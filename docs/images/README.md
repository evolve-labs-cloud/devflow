# Screenshots

Adicione os screenshots aqui:

| Arquivo | Descrição | Tamanho Recomendado |
|---------|-----------|---------------------|
| `hero.png` | Home page com input de projeto | 1600x900 |
| `terminal.png` | Terminal com Quick Actions e agente ativo | 1600x900 |
| `editor.png` | Monaco Editor com arquivo aberto | 800x600 |
| `specs.png` | Specs Panel com progress bars | 800x600 |
| `dashboard.png` | Dashboard com métricas | 800x600 |

## Como capturar

1. Abra a Web IDE: `cd web && npm run dev`
2. Acesse http://localhost:3000
3. Use Cmd+Shift+4 (Mac) ou ferramenta de screenshot
4. Salve os arquivos nesta pasta

## Dicas

- Use resolução 2x para telas Retina
- Mantenha aspect ratio consistente
- Capture com um projeto real carregado
